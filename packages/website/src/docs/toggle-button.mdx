### Custom props

| Name       | Type         |   Default   | Description                                        |
| :--------- | :----------- | :---------: | :------------------------------------------------- |

| `children` | ReactElement |   `null`    | Children should be `Radio` Elements                |
| `variant`  | 'highlight'  | `undefined` | Variant to display                                 |
