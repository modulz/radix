## Examples

### Variants

There are 2 variants to choose from.

```js live
<Avatar variant="gray" alt="Modulz">
  M
</Avatar>

<Avatar variant="black" alt="Modulz" ml={4}>
  M
</Avatar>
```

### Size

There are 2 sizes to choose from.

```js live
<Avatar
  size={0}
  alt="Modulz"
  src="https://avatars0.githubusercontent.com/u/28682402"
>
  M
</Avatar>

<Avatar
  ml={4}
  size={1}
  alt="Modulz"
  src="https://avatars0.githubusercontent.com/u/28682402"
>
  M
</Avatar>
```
