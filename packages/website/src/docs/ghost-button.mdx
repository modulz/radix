## Examples

### size

There are 2 sizes to choose from.

```js live
<GhostButton>
  <PlusIcon  />
</GhostButton>

<GhostButton size={1} ml={6}>
  <HamburgerIcon  />
</GhostButton>
```

### isActive

For when your button is active.

```js live
<GhostButton isActive>
  <PlusIcon />
</GhostButton>
```
