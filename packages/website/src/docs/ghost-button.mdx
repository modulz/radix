### Custom props

| Name       | Type                 | Default  | Description          |
| :--------- | :------------------- | :------: | :------------------- |
| `variant`  | 'normal' \| 'active' | `normal` | Variant to display   |
| `size`     | 0 \| 1               |   `0`    | Size to display      |
| `disabled` | Boolean              | `false`  | Disables the element |
